# PlayerLogger

Advanced Playtime & Command Logging Plugin for Paper 1.21+

## Features

- Logs player join & quit events
- Tracks total playtime (with live session calculation)
- Stores command execution logs
- Tracks online/offline status
- SQLite & MySQL support
- Multi-language support (EN / DE)
- Async database handling
- Crash-safe offline correction
- Top 10 playtime ranking
- Admin tools for managing data

## Commands

/plog stats  
/plog stats <player>  
/plog top  
/plog reload  
/plog delete <player>  
/plog add <player> <time>

## Requirements

- Paper 1.21+
- Java 21

## Database

Supports:
- SQLite (default)
- MySQL / MariaDB
